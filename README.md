# To-Do_Task_Program
-----------------------INSTRUCTIONS-----------------------------------

This is a terminal-based To-Do program written in Python. It was created to learn and implement 
the concepts I have learned by watching Python tutorials online.

Install Dependencies
-- pip install colorama

Run the Program
-- First, navigate to the directory where the program is located
-- python To_Do_Task.py

How to Use the Program
-- When the program runs, it will display a menu with options (Adding a task, Viewing, Removal, etc.)
-- Enter the corresponding number to select a feature
-- The program will store your tasks in To-Do.txt

Menu Options
-- 1 : Add a New task - Create a new task with timestamp
-- 2 : See All Task - View all your tasks
-- 3 : Clear All Task - Delete all tasks (use with caution)
-- 4 : Edit a task - Modify an existing task
-- 5 : Set a task complete - Mark a task as completed with timestamp
-- 0 : Exit - Close the program
